## 📋 Quick Start

### Prerequisites
- Python 3.8+
- Node.js 16+
- Git
- Cashon account for payment processing

### Installation

```bash
# Clone the repository
git clone https://github.com/thealphakenya/Alpha-Q-ai.git
cd Alpha-Q-ai

# Install Python dependencies
pip install -r requirements/ai_automation.txt

# Install Node.js dependencies
npm install

# Setup QMOI environment
node scripts/qmoi-environment-setup.js

# Start QMOI system (Linux/Mac)
./start-qmoi.sh

# Start QMOI system (Windows)
start-qmoi.bat
```

### Master Mode Setup

```bash
# Enable master mode for full revenue control
export QMOI_MASTER_MODE=true
export QMOI_DAILY_REVENUE_TARGET=100000
export QMOI_CASHON_API_KEY=your_cashon_api_key

# Start enhanced auto projects system
node scripts/qmoi-enhanced-auto-projects.js --master-mode
```
