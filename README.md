## Quick Start

### Prerequisites
- Python 3.7+
- Node.js 16+
- Git
- Required API keys (see configuration)

### Installation
```bash
# Clone the repository
git clone https://github.com/qmoi/alpha-q-ai.git
cd alpha-q-ai

# Install dependencies
pip install -r requirements/qmoi_enhanced_requirements.txt
npm install

# Setup environment
cp .env.example .env.local
# Edit .env.local with your API keys

# Start QMOI systems
python scripts/qmoi_auto_startup.py
```

### Configuration
Create `.env.local` with your API keys:
```bash
# GitLab Configuration
GITLAB_ACCESS_TOKEN=your_gitlab_token
GITLAB_URL=https://gitlab.com

# Vercel Configuration
VERCEL_TOKEN=your_vercel_token
VERCEL_PROJECT_ID=your_project_id

# Cashon Configuration
CASHON_MPESA_NUMBER=your_mpesa_number
CASHON_API_KEY=your_cashon_key

# Quantum Configuration
QUANTUM_API_KEY=your_quantum_key
QUANTUM_REVENUE_TARGET=10000
```
